## Features

- Carousel slider to showcase featured movies
- Horizontal scrolling sections for Trending, Popular, and Top Rated movies
- Custom font integration and responsive layout
- Uses local asset images for movie posters
- Static bottom navigation bar for UI demonstration

## Getting Started

### Prerequisites

- Flutter SDK installed (version 3.9.2 or compatible)
- Android Studio or VS Code with Flutter extension
- A connected device or emulator for testing

### Installation

1. Clone the repository:
```
git clone https://github.com/SidqiRaafi/tugas4_movieplayer.git
```

2. Navigate to the project directory:
```
cd tugas4_movieplayer
```

3. Get dependencies:
```
flutter pub get
```

4. Run the app:
```
flutter run
```


### Asset Management

Local images are stored in the `assets/images/` folder and registered in `pubspec.yaml`. Make sure the assets are correctly added and updated before building.

## Usage

This project demonstrates a UI-only Netflix clone example focused on layout and design. It does not include backend or API integrations.
